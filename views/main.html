Original Question
If bier one has 15 beers on tap, how many possible "mixed beers" can you make by combining equal parts from any number of beers in the menu?
Solution It is a classic N choice k problem, where order doesn't matter.

Here is the formula used to compute the number of ways to mix k beers from N total possible beers, where order doesn't matter.

$$\frac{N!}{k!(N-k)!}$$

For example to find the number of ways to mix 3 beers from 15 total beers we use

$$\frac{15!}{3!(15-3)!} = 455$$

To get the total number of combos, you need to sum up all possible combos

$$\sum_{k=1}^N \frac{N!}{k!(N-k)!} = \frac{15!}{1!(15-1)!} + \frac{15!}{2!(15-2)!} + ... + \frac{15!}{15!(15-15)!}$$ Note: by definition 0! = 1
