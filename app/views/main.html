This is a calculator for computing the number of possible combinations of beers

It is a classic n choice k problem, where order doesn't matter. Here is the formula used to compute the number of ways to mix k beers from N total possible beers, where order doesn't matter. $$\frac{N!}{k!(N-k)!}$$ For example to find the number of ways to mix 3 beers from 15 total beers we use $$\frac{15!}{3!(15-3)!} = 455$$ To get the total number of combos, you need to sum up all possible combos $$\sum_{i=0}^N \frac{N!}{k!(N-k)!}$$
